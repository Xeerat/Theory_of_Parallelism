# Sum of sines 

Программа на C++, которая:  
- создаёт массив float или double длиной 10^7 элементов
- заполняет массив значениями синуса одного периода
- считает сумму элементов
- выводит результат в терминал

## Выбор типа массива

Программа поддерживает выбор типа массива во время сборки CMake.

По умолчанию используется `float`.    
Чтобы использовать `double`, необходимо при генерации файлов CMake добавить опцию:   
`-DDOUBLE=ON`

## Примеры вывода

### Float:
```
Array type: float
Sum: 0.349212
```

### Double:
```
Array type: double
Sum: -6.76916e-10
```
